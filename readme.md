# tgPostsNotifierUserBot

### Why?

To notify about pinned posts at any tg channels

### Requirements

1. `Node.JS LTS` (>= v14)
2. `MySQL 5.7`
3. `npm ci` or `npm`
4. Run `schema.sql` and create the row in the `settings` table

### After first run

Write to the bot:

- `/help`
- `/helpCoder`
